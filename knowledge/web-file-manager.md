# Web File Manager

Manage device files from a browser over WiFi.

## Setup (Station Mode)

1. Connect the device to a WiFi network via **WiFi > Network**
2. Download the web page via **WiFi > Network > Download > Web File Manager**
3. Go to **WiFi > Network > Web File Manager**
4. Set a password and start the server
5. Open `http://unigeek.local` or the displayed IP address in a browser

## Setup (Access Point Mode)

1. Go to **WiFi > Access Point**
2. Enable **File Manager** toggle (web files must be installed first)
3. Press **Start**
4. Connect to the AP from another device
5. Visit `http://unigeek.local` — it redirects to the file manager on port 8080
